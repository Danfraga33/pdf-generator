# CV PDF Generator

## Tech stack
- React + Vite 
- Puppeteer for PDF generation

## How to run

- Add your picture in `src/assets/profile.jpeg`
- Fill your information in `data.json`
- Adapt CSS or scale in `pdf.ts` if needed

```sh
npm install 
# Starts the vite server
npm run dev
# Generate the pdf
npm run pdf
```
